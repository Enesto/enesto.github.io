설치하기

.img 파일 오른쪽 클릭 후 연결 프로그램 -> Windows 탐색기

이후 마운트 된 office 드라이브 전체를 복사해서 임의의 장소에 붙여넣기한다.

16.0.10730.20102라는 폴더가 생성된다.

아래 파일을 다운받아 압축을 풀고 안에 wotok.exe 파일을 16.0.10730.20102 폴더 안에 넣는다.

wotok_x64_v2018.11.15.01.zip

wotok.exe를 실행한다.

Visio와 Project를 설치하려면 Install VisioPro2019와 Install ProjectPro2019 체크박스를 체크한다.

커맨드 창을 관리자 권한으로 실행한다. (Win키를 누르고 cmd를 검색하면 나오는 것을 오른쪽으로 클릭 후 관리자 권한으로 실행)

아래 명령중 하나를 실행한다.
